# Подводная лодка-сканер с 360° обзором и построением 2D-карты

![Arduino](https://img.shields.io/badge/Arduino-00979D?style=for-the-badge&logo=Arduino&logoColor=white)
![PlatformIO](https://img.shields.io/badge/PlatformIO-FF7F00?style=for-the-badge&logo=platformio&logoColor=white)
![License](https://img.shields.io/badge/license-MIT-green)

Проект представляет собой автономный или управляемый по проводу подводный аппарат (ROV), способный сканировать окружающее пространство на 360° и формировать двумерную карту дна/препятствий. Используется недорогой ультразвуковой сонар, вращающийся с помощью сервопривода или шагового двигателя. Данные обрабатываются на Arduino и передаются на ПК для визуализации в реальном времени.

![Пример карты](https://via.placeholder.com/600x400?text=Пример+2D+карты+сканирования)

## Возможности

- **Круговое сканирование** – поворот сонара на 360° с заданным шагом.
- **Построение 2D-карты** – преобразование полярных координат (расстояние, угол) в декартовы и отображение препятствий.
- **Режимы работы** – однократное сканирование, непрерывное циклическое сканирование, запись на SD-карту.
- **Глубиномер** (опционально) – датчик давления для привязки карты к глубине.
- **Передача по UART** – данные отправляются на ПК в текстовом или бинарном виде.
- **Визуализация на Python** – скрипт для приёма данных, фильтрации и построения карты в реальном времени.

## Аппаратная часть

| Компонент | Назначение | Пример модели |
|-----------|------------|----------------|
| Arduino | Главный контроллер | Arduino Uno / Nano / Mega |
| Ультразвуковой сонар | Измерение расстояния до препятствий | JSN-SR04T (герметичный) / Ping360 |
| Сервопривод или шаговый двигатель | Поворот сонара на 360° | MG995 (с доработкой для воды) + герметичный сальник |
| Датчик давления (опционально) | Измерение глубины | MS5837 / BMP280 (в маслонаполненной капсуле) |
| SD-карта (опционально) | Логирование данных | Модуль MicroSD (SPI) |
| Аккумулятор | Питание | Li-Ion 12V / 7.4V |
| Герметичный корпус | Защита электроники | ПВХ трубы, напечатанный на 3D-принтере корпус |

### Схема подключения (пример для Arduino Uno + серво + JSN-SR04T)
JSN-SR04T:
VCC -> 5V Arduino
GND -> GND
Trig -> D9
Echo -> D10

Сервопривод (поворотный):
VCC -> 5V (через внешний источник)
GND -> GND Arduino
PWM -> D11

SD-карта (SPI):
CS -> D4
MOSI -> D11 (на Uno – D11)
MISO -> D12
SCK -> D13

Для передачи на ПК – USB-кабель или Bluetooth-модуль (HC-05).

## Программная часть

### Arduino Sketch

- Сканирование выполняется циклически: серво устанавливается в угол `angle`, делается несколько измерений расстояния, усреднение, отправка через Serial.
- Данные передаются в виде: `<угол>;<расстояние>` или с дополнительной информацией.
- Для уменьшения шума можно применять медианный фильтр.
- Пример вывода в Serial:
0;245
45;312
90;178
...

### Python скрипт для визуализации

- Принимает строки из последовательного порта.
- Преобразует полярные координаты в декартовы.
- Отображает точки на декартовой сетке (matplotlib) или обновляет изображение в реальном времени (pygame, OpenCV).
- Реализована опция сохранения карты в PNG.

## Установка и запуск

### 1. Сборка аппаратной части

- Поместите электронику в герметичный корпус. Сонар закрепите на вращающейся платформе с сальником.
- Обеспечьте водонепроницаемость всех проходов (эпоксидная смола, силикон).
- Используйте внешний BEC для питания сервопривода, чтобы не перегружать Arduino.

### 2. Загрузка прошивки

- Установите Arduino IDE или PlatformIO.
- Откройте файл `submarine_scanner.ino`.
- Настройте пины в соответствии со схемой.
- Загрузите скетч в плату.
